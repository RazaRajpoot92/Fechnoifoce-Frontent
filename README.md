# Techno I Force

**Techno I Force** is a modern, responsive automotive website built with **React** and **Tailwind CSS**.  

## Features
- Professional theme with smooth gradients  
- Interactive UI with hover effects and transitions  
- Modern fonts: Inter (body) & Rajdhani (headings)  
- Fully responsive and mobile-friendly  

## Tech Stack
React | Tailwind CSS | Vite
